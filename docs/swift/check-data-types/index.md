# Check Data Types

Data types for the Check namespace.

## EchoArg

Contains the arguments to be sent to the Dropbox servers.

**Properties:**
- `query: String` - The string that you'd like to be echoed back to you.

---

## EchoResult

EchoResult contains the result returned from the Dropbox servers.

**Properties:**
- `result: String` - If everything worked correctly, this would be the same as query.
